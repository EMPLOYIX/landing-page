# Employix — Landing Page

Static landing page for [employix.co.in](https://employix.co.in), served via GitHub Pages.

## Structure

- `index.html` — the full single-page site (styles inlined, no build step)
- `favicon.svg` — brand mark
- `robots.txt` — crawler directives
- `CNAME` — custom domain binding for GitHub Pages

## Local preview

Open `index.html` in a browser, or run any static server from this directory.

## Deploy

Pushing to `main` publishes automatically through GitHub Pages.
